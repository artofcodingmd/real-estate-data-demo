# Real Estate Web API Demo
ASP.NET Core backend project using GitHub Codespaces,
SQL Server via Docker, AWS SDK integration, and Swagger-based data visualization with seeded sample data. Project created following developer prompted AI generated tutorial.

### Running in GitHub Codespaces

This project supports one-click startup using GitHub Codespaces.
- SQL Server runs locally via Docker
- AWS services authenticate via injected environment variables
- No local setup or credentials are required

This mirrors how the application runs in cloud-hosted environments.
